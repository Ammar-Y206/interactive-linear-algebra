# Interactive Linear Algebra

An interactive, visual-first journey through Linear Algebra inspired by
the teaching philosophy of **3Blue1Brown**, **Brilliant**, and
**Duolingo**.

Instead of treating Linear Algebra as a collection of formulas, this
project teaches it as a language for describing space, movement, and
computation. Every lesson is designed around intuition, simulations,
thought experiments, and real-world applications.

## Live Demo

**Website:**
https://interactive-linear-algebra-gthngjhet-aym1352006-4407s-projects.vercel.app

## Features

-   22 carefully structured lessons
-   Interactive simulations for core concepts
-   Visual explanations before formal definitions
-   Concept checks and quizzes
-   Progress tracking
-   Search and glossary support
-   Responsive UI built with Next.js and Tailwind
-   Motivational "Why does this matter?" sections
-   AI, robotics, graphics, and data science applications

## Curriculum Overview

### Introductory Arc

1.  Welcome to Linear Algebra
2.  Why Learn Linear Algebra?
3.  Linear Algebra in Real Life
4.  The Hidden Math Behind AI
5.  Mathematics as a Language
6.  The Roadmap of This Course

### Core Topics

-   Vectors
-   Span and Linear Combinations
-   Basis and Linear Dependence
-   Transformations
-   Matrix Composition
-   Determinants
-   Inverse Matrices
-   Dot Product
-   Cross Product
-   Eigenvectors & Eigenvalues
-   Change of Basis
-   Cramer's Rule
-   Abstract Vector Spaces

## Teaching Philosophy

> "Do not think of this project as a course. Think of it as building the
> Duolingo/Brilliant of Linear Algebra."

Each lesson follows: - Hook - Intuition - Interactive Simulation -
Concept Check - Connections
